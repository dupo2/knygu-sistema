# KnyguSistema

1. Firebase Realtime DB: Realizuotas duomenų saugojimas (title, pages, isbn, short_description, authors[]).
2. UI/CRUD: Sukurta vartotojo sąsaja knygų pridėjimui, redagavimui ir trynimui. Sąrašas automatiškai rikiuojamas pagal pavadinimą.
3. Validacija: Įdiegta patikra, neleidžianti įkelti knygos su egzistuojančiu ISBN.
4. Filtravimas: Realizuota paieška pagal pavadinimą ir ISBN numerį.
5. Autentifikacija: Sukurta prisijungimo sistema. Svečiai gali tik peržiūrėti sąrašą, o administravimo teisės suteikiamos tik registruotiems vartotojams.
